# 百度

用途与边界

- 面向公开官网、产品页、帮助中心、开放平台与公告页的信息整理
- 不提供搜索作弊、收录规避、后台操作、账号代管或权限绕过能力
- 仅处理公开可见的搜索入口、产品能力、生态服务与帮助信息

关键入口

- 主页：https://www.baidu.com/
- 搜索与生态：搜索主页、产品矩阵、站长或开放平台入口
- 帮助与公告：帮助中心、规则页、公告页、产品说明入口

常见任务

- 搜索产品矩阵、内容入口与生态服务整理
- 站长支持、开放能力、帮助页与规则说明提取
- 产品更新、公告与公开资源入口汇总

数据字段

- 产品名称、服务入口、开放能力、链接
- 规则页、帮助页、公告页、站长入口链接
- 地区差异、支持内容类型、FAQ

自动化要点

- 搜索相关规则与产品线常有拆分，整理时按入口分类展示
- 不输出作弊、刷量、快排、规避规则的建议
- 不处理站长后台、广告账户或私有搜索数据

示例流程

- 入口整理：访问 Baidu 主页与产品页 → 输出服务矩阵
- 规则汇总：访问帮助与规则页 → 提取关键限制 → 输出清单
- 生态能力：进入开放平台 → 提取服务说明与入口 → 输出摘要

合规提示

- 遵守平台规则、版权要求与账号安全要求
- 只整理公开信息，不输出敏感数据或非公开内容
