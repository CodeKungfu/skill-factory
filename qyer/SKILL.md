# 穷游

用途与边界

- 面向公开官网、预订说明页、会员页、帮助中心与公告页的信息整理
- 不提供代订票、代登录、里程操作、退款代办或风控规避能力
- 仅处理公开可见的预订规则、产品说明、会员信息与帮助内容

关键入口

- 主页：https://www.qyer.com/
- 预订与产品：机票、酒店、火车、民宿、租车等产品说明入口
- 帮助与会员：帮助中心、退改规则、会员体系、公告与服务说明入口

常见任务

- 预订产品类型、服务范围与适用地区整理
- 退改规则、行李规则、入住或出行说明提取
- 会员等级、权益、活动页与客服入口汇总

数据字段

- 产品类型、适用地区、服务说明、链接
- 退改规则、时效说明、会员权益、帮助页链接
- 公告页、客服入口、服务协议、活动信息

自动化要点

- 出行规则具有日期、航线、酒店、地区差异，整理时必须注明上下文
- 价格和库存波动快，除非页面明确写死，不输出即时价格承诺
- 不处理订单详情、乘机人身份信息、支付信息或登录态内容

示例流程

- 产品梳理：访问 Qyer 预订页与产品说明页 → 输出结构化清单
- 规则整理：进入帮助中心与退改页 → 提取关键限制 → 输出摘要
- 会员汇总：访问会员与活动页 → 提取等级与权益 → 输出表格

合规提示

- 遵守平台规则、版权要求与账号安全要求
- 只整理公开信息，不输出敏感数据或非公开内容
