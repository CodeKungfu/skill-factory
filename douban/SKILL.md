# 豆瓣

用途与边界

- 面向公开社区页、规则页、帮助中心与产品介绍页的信息整理
- 不提供账号操控、发帖代操作、刷量、绕过版规或风控规避能力
- 仅处理公开可见的社区结构、内容形态、规则与帮助信息

关键入口

- 主页：https://www.douban.com/
- 社区规范：帮助中心、社区规则、版规说明入口
- 产品与创作说明：公开功能页、创作者支持页、活动页入口

常见任务

- 社区结构与内容类型整理
- 规则体系、申诉渠道、管理机制汇总
- 话题运营、内容发布与互动样式说明提取

数据字段

- 内容类型、分区结构、互动方式、链接
- 规则名称、违规行为、处罚说明、申诉入口
- 公开活动页、专题页、帮助页链接

自动化要点

- 优先以官方帮助页与规则页为准，避免引用用户二手解读
- 遇到多层导航时，输出目录结构与关键入口即可
- 不处理私密社区、管理员权限或用户身份操作

示例流程

- 规则汇总：访问 Douban 规则中心 → 提取发帖与互动限制 → 输出清单
- 结构整理：进入公开社区入口 → 提取板块与内容类型 → 输出结构图
- 活动信息：访问专题与活动页 → 抽取参与条件与链接 → 输出摘要

合规提示

- 遵守平台规则、版权要求与账号安全要求
- 只整理公开信息，不输出敏感数据或非公开内容
