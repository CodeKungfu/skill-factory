# 喜马拉雅

用途与边界

- 面向公开官网、专辑页、会员页、创作者页与帮助中心的信息整理
- 不提供账号共享、版权绕过、音频抓取、接口调用或付费规避能力
- 仅处理公开可见的内容入口、会员说明、创作者支持与帮助信息

关键入口

- 主页：https://www.ximalaya.com/
- 内容与创作：专辑、主播、分类、创作者平台入口
- 会员与帮助：会员页、权益说明、帮助中心、版权页入口

常见任务

- 内容分类、专辑入口、主播入口与播放场景整理
- 会员权益、下载与设备说明、创作者支持汇总
- 帮助中心、版权说明、FAQ 与活动页提取

数据字段

- 内容分类、专辑入口、会员权益、链接
- 设备支持、下载规则、帮助页、版权页链接
- 创作者入口、活动页、FAQ、套餐说明

自动化要点

- 音频内容和权益可能按地区变化，整理时注明适用范围
- 不输出内容抓取、版权绕过或账号共享建议
- 不处理个人收藏、收听记录、创作者后台或私有专辑数据

示例流程

- 入口整理：访问 Ximalaya 分类与专辑页 → 输出内容结构
- 权益汇总：进入会员页 → 提取下载与设备规则 → 输出表格
- 帮助摘要：访问帮助中心与版权页 → 输出关键说明

合规提示

- 遵守平台规则、版权要求与账号安全要求
- 只整理公开信息，不输出敏感数据或非公开内容
