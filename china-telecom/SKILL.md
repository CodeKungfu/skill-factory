# 中国电信

用途与边界

- 面向公开官网、套餐页、业务办理页、帮助中心与公告页的信息整理
- 不提供实名绕过、代办开卡、后台操作、设备解锁或风控规避能力
- 仅处理公开可见的套餐、宽带、增值服务、业务规则与帮助信息

关键入口

- 主页：https://www.189.cn/
- 套餐与业务：套餐页、宽带页、业务办理、终端服务入口
- 帮助与公告：帮助中心、资费说明、公告页、客服入口

常见任务

- 套餐结构、宽带服务、流量规则与资费说明整理
- 业务办理入口、实名要求、客服与帮助信息汇总
- 公开公告、地区差异、设备服务与 FAQ 提取

数据字段

- 套餐名称、资费、流量规则、链接
- 地区说明、办理入口、帮助页、客服链接
- 实名要求、公告页、宽带与增值服务说明

自动化要点

- 通信套餐强依赖地区和时间，整理时必须标注省市或适用范围
- 不得输出实名规避、黑卡、解锁、薅资费漏洞等建议
- 不处理号码状态、通话记录、实名信息或私有账单数据

示例流程

- 套餐整理：访问 China Telecom 套餐页 → 提取资费与限制 → 输出对照表
- 业务梳理：进入业务办理页 → 提取宽带与增值服务入口 → 输出清单
- 帮助摘要：访问公告与帮助中心 → 提取实名、客服、FAQ → 输出摘要

合规提示

- 遵守平台规则、版权要求与账号安全要求
- 只整理公开信息，不输出敏感数据或非公开内容
