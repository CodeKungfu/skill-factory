# ChatGPT

用途与边界

- 面向公开产品页、模型页、文档中心与公告页的信息整理
- 不提供账号代操作、付费订阅管理、接口调用或权限绕过能力
- 仅处理公开可见的模型、功能、套餐、版本与文档信息

关键入口

- 主页：https://chatgpt.com/
- 模型与功能：产品页或能力页入口
- 定价与文档：官网导航中的定价、文档、更新日志入口

常见任务

- 模型能力摘要（上下文、模态、工具能力、典型场景）
- 套餐与价格说明整理（免费版、付费版、额度与限制）
- 文档目录、快速开始、示例链接汇总

数据字段

- 产品名称、模型名称、版本、能力标签、上下文长度、链接
- 套餐名称、价格、额度、功能差异、适用对象、链接
- 文档章节、示例入口、发布时间、公告链接

自动化要点

- 页面可能动态加载，抓取前需要等待主内容完成渲染
- 优先保留原始页面链接，避免二次转述丢失条件约束
- 不执行 API 调用，不处理密钥、账号、企业后台配置

示例流程

- 能力摘要：访问 ChatGPT 公开能力页 → 抽取模型与特性 → 输出结构化摘要
- 定价整理：进入 ChatGPT 定价页 → 提取套餐、额度、限制 → 输出对照表
- 文档汇总：进入 ChatGPT 文档中心 → 抽取目录与示例入口 → 输出链接清单

合规提示

- 遵守平台规则、版权要求与账号安全要求
- 只整理公开信息，不输出敏感数据或非公开内容
