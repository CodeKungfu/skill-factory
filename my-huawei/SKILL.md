# 我的华为

用途与边界

- 面向公开官网、服务页、社区页、活动页与帮助中心的信息整理
- 不提供售后代办、账号接管、保修绕过、刷机破解或权限绕过能力
- 仅处理公开可见的设备服务、会员活动、售后说明与社区信息

关键入口

- 主页：https://consumer.huawei.com/cn/community/
- 服务与社区：服务中心、社区入口、会员活动、售后说明入口
- 帮助与公告：帮助中心、保修政策、更新公告、设备支持入口

常见任务

- 设备服务入口、会员活动、售后能力与帮助信息整理
- 保修规则、服务网点、社区活动与支持文档汇总
- 公开公告、版本信息、服务政策提取

数据字段

- 服务入口、设备支持、保修说明、链接
- 社区活动、帮助页、公告页、政策链接
- 会员权益、网点查询、FAQ、设备要求

自动化要点

- 品牌服务常按设备型号、地区、保修状态区分，整理时应明确上下文
- 不提供刷机、解锁、破解、保修规避或售后造假建议
- 不处理用户设备序列号、保修单据、账号资料或私有工单

示例流程

- 服务整理：访问 My Huawei 服务页 → 提取入口与规则 → 输出清单
- 活动汇总：进入社区与会员页 → 提取活动与权益 → 输出摘要
- 帮助梳理：访问帮助中心 → 输出售后与设备支持目录

合规提示

- 遵守平台规则、版权要求与账号安全要求
- 只整理公开信息，不输出敏感数据或非公开内容
