# 百度地图

用途与边界

- 面向公开官网、地点服务页、商家合作页、帮助中心与产品说明页的信息整理
- 不提供实时定位跟踪、账号操作、商家后台修改或风控规避能力
- 仅处理公开可见的导航功能、地点服务、商家能力与帮助信息

关键入口

- 主页：https://map.baidu.com/
- 地点与商家服务：商户入驻、地点管理、地图服务说明入口
- 帮助与产品：帮助中心、导航说明、开放平台或产品能力入口

常见任务

- 导航能力、地点服务、地图产品说明汇总
- 商家入驻、地点认领、信息维护与帮助规则整理
- 开放平台、API 说明入口、地图更新与纠错入口汇总

数据字段

- 导航功能、地点服务、商家入口、链接
- 帮助页、纠错入口、开放平台、产品说明链接
- 地区支持、设备要求、使用限制

自动化要点

- 地图服务高度依赖地区和终端系统，输出时标注适用平台
- 地点信息和路线会实时变化，优先整理规则与入口，而非断言静态结果
- 不处理实时定位、个人轨迹、商家后台或私有地址数据

示例流程

- 功能整理：访问 Baidu Maps 产品页 → 抽取导航与地点能力 → 输出摘要
- 商家说明：进入商家或地点服务页 → 提取认领与维护规则 → 输出清单
- 帮助汇总：访问帮助中心 → 提取纠错、开放平台与常见问题入口

合规提示

- 遵守平台规则、版权要求与账号安全要求
- 只整理公开信息，不输出敏感数据或非公开内容
