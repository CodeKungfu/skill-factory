# 猎聘

用途与边界

- 面向公开官网、企业招聘页、职位规则页、帮助中心与服务页的信息整理
- 不提供账号代招、简历爬取、候选人骚扰、风控规避或数据售卖能力
- 仅处理公开可见的职位发布规则、企业服务、帮助信息与产品说明

关键入口

- 主页：https://www.liepin.com/
- 企业与招聘服务：企业招聘页、职位发布、雇主品牌、套餐说明入口
- 帮助与规则：帮助中心、违规说明、用户协议、客服入口

常见任务

- 职位发布规则、企业服务、套餐与流程整理
- 候选人触达、雇主品牌、招聘工具与帮助信息汇总
- 违规限制、审核规则、客服与申诉入口提取

数据字段

- 企业服务、职位发布规则、套餐说明、链接
- 帮助页、审核规则、客服入口、协议链接
- 适用行业、城市范围、限制条件、FAQ

自动化要点

- 招聘规则存在地区和岗位类别差异，整理时注明适用范围
- 不得输出骚扰式触达、数据抓取、批量刷岗位或规避审核建议
- 不处理企业后台、候选人简历、联系方式或私有招聘数据

示例流程

- 规则整理：访问 Liepin 企业页与帮助中心 → 提取职位发布规则 → 输出清单
- 服务汇总：进入套餐与产品页 → 提取企业能力与差异 → 输出表格
- 帮助摘要：访问帮助页 → 提取客服、审核、申诉入口 → 输出摘要

合规提示

- 遵守平台规则、版权要求与账号安全要求
- 只整理公开信息，不输出敏感数据或非公开内容
